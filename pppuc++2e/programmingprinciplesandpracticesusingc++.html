3.5 Assignment and initialization

int a = 3; // a gets the value 3
           // solution: a starts out with the value 3
a = 4;     // a becomes 4
           // solution: a gets the value 4("becomes 4") 
int b = a; // b = a(a is 4)
           // solution: b starts out with a copy of a's value
b = a + 5; // 4 = 4 + 5 (b now equals 9)
           // solution: b gets the value a+5(this,9)
a = a + 7  // 4 = 4 + 7 (a: 11)
           // a gets the value a+7 (that is, 11)
a = a + 7
1) first, get the value of a; thats the integer 4
2) next, add 7 to that 4, yielding the integer 11
3) finally, oyt that 11 into a

3.8 Types and objects

A type defines a set of possible values and a set of operations(for an object).
An object is some memory that holds a value of a given type.
A value is a set of bits in memory that holds a value of a given type.
A variable is a named object.
A declaration is a statement that gives a name to an object.
A definition is a declaration that sets aside memory for an object.
